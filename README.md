# Tactics: Isometric Turn-Based Strategy (C++ / SDL2)

Written by Me: SeNiah McField

A turn-based isometric strategy prototype written in C++ using SDL2.  
Features: isometric tile map rendering, A* pathfinding, turn manager, simple AI, UI buttons, sprite atlas-based animations.

This project is inspired by the turn-based logic of Tactical Breach Wizards, this C++ prototype allowed me to explore isometric engine architecture and grid-based systems. It features a hand-coded A pathfinding* system, a JSON Atlas Parser for data-driven animations, and the Painter’s Algorithm for depth sorting. This project represents a major milestone in my growth as a programmer. It challenged me to refresh my mind on manual memory management and coordinate transformation math, resulting in a leak-free and performant foundation for tactical gameplay. I am incredibly proud of the code and output of this project and hope you all enjoy reviewing it!
